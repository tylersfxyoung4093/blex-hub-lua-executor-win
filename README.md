## FAQ

**Is Blex Hub only for Windows?**  
Yes, the project is framed as a Windows-first desktop experience, with Windows 10 and 11 as the main supported environments.

**Will scripts still work after Roblox updates?**  
That can change whenever Roblox or an executor changes. Actual behavior depends on the current game build and the executor setup being used.

**How is Blex Hub different from a standalone executor?**  
Blex Hub acts as the script hub and collection layer; running scripts still requires a compatible executor.

**Does using scripts affect my account?**  
Using scripts can carry platform and game-rule risks. Check the relevant terms and proceed at your own judgment.

**Are scripts stored locally?**  
The hub layout implies local organization for scripts and related files, which helps keep content easier to manage on your device.
